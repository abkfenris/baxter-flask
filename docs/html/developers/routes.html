HTTP Routes for site¶

Browseable routes¶

GET /avalanche/incident/¶

Show all incidents

GET /avalanche/path/¶

Show all avalanche paths

GET /trails/¶

Show all trails

GET /¶

Index page for site.

GET /avalanche/incident/(int: id)¶

Show a single incident

GET /avalanche/path/(id)/¶

Show a single trail

GET /trails/(id)/¶

Show a single trail

API routes¶

GET /api/1.0/avalanche/incident/¶

Geojson for all avalanche incidents

GET /api/1.0/avalanche/path/¶

Geojson for all avalanche paths

GET /api/1.0/trail/¶

Show all trails

GET /api/1.0/avalanche/incident/(int: id)/parts¶

return geojson for individual avalanche incident

GET /api/1.0/avalanche/incident/(int: id)/¶

return geojson for individual avalanche incident

GET /api/1.0/avalanche/path/(int: id)/¶

return geojson for individual avalanche path

GET /api/1.0/trail/(int: id)/¶

Show individual trail id
